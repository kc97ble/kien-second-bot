# Database

## Table `movements`

```
Movement: object {
    date: string;
    author: string;
    action: string;
    item: string;
}
```

## Table `states`

```
State: object {
    author: string;
    action: string; // "", "borrow", "return"
}
```

## Table `stages`

```
Stage: object {
    date: string;
    author: string;
    action: string;
    item: string;
}
```
